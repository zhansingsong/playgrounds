关于z-index的一些事

一般z-index失效基本上都是不了解z-index是如何工作导致的（这没有看这篇blog之前，至少我也是这样的）。它不复杂，但如果没有花时间去查看z-index的规范文档，可能会不了解一些关键的技术点。
不信？能否解答如下问题，作为检验你是否真正了解z-index：

问题：

HTML结构如下：

<div>
  <span class="red">Red</span>
</div>
<div>
  <span class="green">Green</span>
</div>
<div>
  <span class="blue">Blue</span>
</div>

CSS如下：

.red, .green, .blue {
  position: absolute;
}
.red {
  background: red;
  z-index: 1;
}
.green {
  background: green;
}
.blue {
  background: blue;
}
